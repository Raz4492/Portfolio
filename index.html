Advancing Bangla Machine Translation Through Informal Datasets

This thesis explores advancing Bangla machine translation using informal datasets. Leveraging cutting-edge NLP techniques such as LSTM, BiLSTM, and BERT (mT5-large), the study evaluates translation quality using BLEU metrics.
